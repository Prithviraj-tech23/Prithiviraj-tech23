IoT-Enabled Smart Dustbin

The Problem: Public waste bins often overflow, creating unhygienic conditions and inefficient manual waste sorting.

My Solution: I designed and built a prototype of a smart dustbin that uses sensors to automate and improve waste management. The goal was to create a cleaner, more efficient system.

Key Features:

  • Automatic Waste Segmentation: Utilizes sensors to identify and sort different types of waste into separate compartments.
  • Auto-Lid Function: Detects when the bin is full and automatically closes the lid to prevent overflow and maintain hygiene.
  • User-Friendly Override: Equipped with a simple push-button for users to open the lid manually when needed.

Technologies Used:

Arduino, C++, Ultrasonic Sensors, Servo Motors, Custom Hardware Assembly.

Health Website UI/UX Design

The Goal: To design a clean, intuitive, and accessible user interface for a health-focused website. The design needed to be trustworthy and easy for users of all ages and technical abilities to navigate.

My Process: I started with user research and created personas to understand the target audience. From there, I developed wireframes to map out the user flow and information architecture. Finally, I created high-fidelity mockups and a clickable prototype.

Key Features of the Design:

  • Clean & Minimalist Layout: Focused on readability and reducing clutter to make information easy to find.
  • Accessible Color Palette: Used a color scheme with high contrast ratios to ensure readability for visually impaired users.
  • Intuitive Navigation: Designed a simple and clear navigation structure to help users find doctors, articles, and services quickly.

Tools Used:

Figma, Wireframing, Prototyping, User Persona Development.

RFID Attendance System

The Problem: Manual attendance tracking is time-consuming, prone to errors, and inefficient for large groups.

My Solution: I designed and implemented an automated attendance checker using an RFID kit. Each user is given an RFID card, and they simply tap it on a reader to have their attendance recorded instantly.

Key Features:

  • Instant & Accurate Tracking: Records attendance in real-time with no manual entry required.
  • Simple User Interaction: Users just need to tap their card, making the process quick and seamless.
  • Data Logging: The system logs attendance data with timestamps for easy management and reporting.

Technologies Used:

Arduino, RFID-RC522 Sensor, C++, Data Logging to SD Card/Serial Monitor.
